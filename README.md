# KlirTechChallenge

## Compatibility
- #### Node v14.18.3
- #### Npm v6.14.1
- #### .NetCore v3.1.426

## How to Run
- Go to `\KlirTechChallenge\angular`
- Then `npm start`
- Go to `\KlirTechChallenge` and run `dotnet restore`
- Go to `\KlirTechChallenge\aspnet-core\Klir.TechChallenge.Web.Api` and run `dotnet watch run`
